# SOC Home Lab

## Overview

This project demonstrates a Security Operations Center (SOC) home lab built using virtualization, SIEM monitoring, attack simulation, and detection engineering techniques.

The lab is designed to simulate real-world enterprise attack and defense scenarios for cybersecurity learning and SOC analyst training.

---

## Objectives

- Practice SOC analyst workflows
- Simulate cyber attacks
- Analyze logs and alerts
- Build detection rules
- Learn incident response
- Improve threat detection skills

---

## Technologies Used

| Technology | Purpose |
|---|---|
| Proxmox VE | Virtualization platform |
| Security Onion | SIEM and monitoring |
| Kali Linux | Attacker machine |
| Windows 11 | Victim machine |
| Sysmon | Endpoint logging |
| Suricata | Network IDS |
| Elastic Stack (ELK) | Log analysis |
